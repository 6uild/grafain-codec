@6uild/grafain

This package is the Grafain implementation of the BlockchainConnection interface. It contains the codec and types to interact with the blockchain backend.

The main entry point is the GrafainConnection, which creates a Tendermint client (from iov-tendermint-rpc) and a codec to parse transactions. The Grafain codec for reading and writing transactions is exported as top-level GrafainCodec.
